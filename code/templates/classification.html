Classification Index

From 1823 to 1985 the British Library used a topical index of pressmarks indicating the physical arrangement of books on shelves in the library. This classification index is arranged in several levels of granularity, from the top level main classification (e.g. "Fiction", "Geography", "Medicine") to more fine-grained topics (e.g. "Geography" → "Ireland and Scotland"). Using a combination of OCR text recognition and manual annotation, we have digitised the higher levels of the index.
