Semantic Network Visualisation

To explore relationships between words and concepts in the British Library Nineteenth Century Digitised Books Collection (BL19), this tool creates and visualises semantic networks. Each node (circle) represents a word or concept, and each edge (link) between two nodes indicates a relationship. To create a network, enter one or more seed words below and click ‘Visualise’. Double-click a node to view the relevant search results from the BL19 collection. Use the navigation controls, or scroll, to pan and zoom. Use the export buttons below the network to save the visualisation as a PNG image or download the network data as a GEXF file for use in tools such as Gephi.
