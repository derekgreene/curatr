Semantic Network Visualisation

To explore relationships between words and concepts in the British Library Nineteenth Century Digitised Books Collection (BL19), this tool creates and visualises semantic networks. In such a network, each node (circle) represents a word, and each edge (link) between two nodes indicates a semantic association between those words.

To create a network, enter one or more seed words below and click "Visualise". Seed words appear as yellow nodes, and associated words as blue nodes. Double-click a node to view the corresponding search results.
